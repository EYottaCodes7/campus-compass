# Campus Compass

Campus Compass is a student-friendly university navigation app for finding important places on campus quickly.

## What it does

- Finds shopping spots, restaurants, lost-and-found desks, support centers, and restricted-access offices
- Lets students filter by category, budget, walking distance, access rules, open status, and accessibility
- Supports route planning from different starting points on campus
- Shows a visual campus navigation board with clickable spot pins
- Includes favorites, notices, route steps, and detailed place information

## Run

Open `index.html` in a browser.

Or run a local dev server (recommended):

```bash
npm install
npm run dev
```

## Project files

- `index.html` - app layout and UI sections
- `styles.css` - styling, responsive layout, and visual theme
- `app.js` - campus data, filters, favorites, route planner, and rendering logic
